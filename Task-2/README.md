## 🛠️ Technologies & Libraries Used

**🐍 Python**

Python was used as the programming language because of its simplicity, readability, and powerful support for Data Science and Artificial Intelligence.

**📊 Pandas**

Pandas was used to load, organize, clean, and analyze the dataset. It made it easy to work with tabular data and perform operations such as filtering, grouping, and calculating statistics.

**🔢 NumPy**

NumPy was used for numerical computations and mathematical operations, helping perform calculations efficiently on the dataset.

**📈 Matplotlib**

Matplotlib was used to create basic data visualizations such as Bar Charts, Pie Charts, Histograms, Line Charts, and Scatter Plots.

**🎨 Seaborn**

Seaborn was used to create more attractive and informative statistical visualizations. It provides enhanced styling and works seamlessly with Pandas DataFrames.

**💻 Google Colab**

Google Colab was used as the development environment to write, execute, and test the Python code without requiring any local software installation.
