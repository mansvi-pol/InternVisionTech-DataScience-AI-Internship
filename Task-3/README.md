## 🛠️ Technologies & Libraries Used

### Programming Language

- **Python** : Python was used as the programming language because of its simplicity, readability, and powerful support for Data Science and Artificial Intelligence.

### Libraries

- **Pandas** : Pandas was used to load, organize, clean, and analyze the dataset. It made it easy to work with tabular data and perform operations such as filtering, grouping, and calculating statistics.
  
- **NumPy** : NumPy was used for numerical computations and mathematical operations, helping perform calculations efficiently on the dataset.
  
- **Matplotlib** : Matplotlib was used to create basic data visualizations such as Bar Charts, Pie Charts, Histograms, Line Charts, and Scatter Plots.
  
- **Seaborn** : Seaborn was used to create more attractive and informative statistical visualizations. It provides enhanced styling and works seamlessly with Pandas DataFrames.

### Development Environment

- **Google Colab** : Google Colab was used as the development environment to write, execute, and test the Python code without requiring any local software installation.

## 📈 Visualizations Created

The dashboard includes the following visualizations:

- **Bar Chart** – Compares total sales across different product categories.
- **Pie Chart** – Shows the percentage distribution of sales by region.
- **Histogram** – Displays the distribution of sales values across orders.
- **Box Plot** – Identifies the spread of profit values and detects potential outliers.
- **Scatter Plot** – Analyzes the relationship between Sales and Profit.
- **Correlation Heatmap** – Visualizes the correlation between numerical features in the dataset.
- **Monthly Sales Trend** – Illustrates how sales changed over different months.
- **KPI Summary Dashboard** – Presents key business metrics such as Total Sales, Profit, Orders, and Average Discount.
- **Business Dashboard** – Combines multiple visualizations into a single dashboard for an overall view of business performance.
