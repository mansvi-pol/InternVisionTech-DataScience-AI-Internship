# Task 1: Data Analysis & AI Fundamentals Using Python

## 📌 Objective

The objective of this task was to understand the fundamentals of Data Science and Artificial Intelligence by performing data analysis on a student dataset using Python. The task focused on data exploration, statistical analysis, and visualization to extract meaningful insights from the data.

## 🏢 Internship

**Organization:** InternVision Tech

## 📂 Dataset

A sample dataset containing **50 student records** was created for this task.

### Dataset Fields

- Student ID
- Student Name
- Department
- Attendance Percentage
- Marks Obtained
- Study Hours
- Semester

The dataset was used to perform various data analysis operations and visualize the relationships between different attributes.

---

## 🛠️ Technologies & Libraries Used

### 🐍 Python
Python was used as the programming language because of its simplicity, readability, and powerful support for Data Science and Artificial Intelligence.

### 📊 Pandas
Pandas was used to load, organize, clean, and analyze the dataset. It made it easy to work with tabular data and perform operations such as filtering, grouping, and calculating statistics.

### 🔢 NumPy
NumPy was used for numerical computations and mathematical operations, helping perform calculations efficiently on the dataset.

### 📈 Matplotlib
Matplotlib was used to create basic data visualizations such as Bar Charts, Pie Charts, Histograms, Line Charts, and Scatter Plots.

### 🎨 Seaborn
Seaborn was used to create more attractive and informative statistical visualizations. It provides enhanced styling and works seamlessly with Pandas DataFrames.

### 💻 Google Colab
Google Colab was used as the development environment to write, execute, and test the Python code without requiring any local software installation.

---

## 📊 Data Analysis Performed

The following data analysis operations were performed:

- Loaded the dataset using Pandas
- Displayed all records of the dataset
- Displayed dataset information
- Checked for missing values
- Generated summary statistics
- Calculated the average marks
- Found the highest and lowest marks
- Performed department-wise performance analysis
- Analyzed the relationship between Attendance Percentage and Marks Obtained
- Analyzed the relationship between Study Hours and Marks Obtained
- Identified the Top 10 Performing Students

---

## 📈 Data Visualizations

The following visualizations were created to better understand the dataset:

- 📊 Bar Chart
- 🥧 Pie Chart
- 📉 Histogram
- 📈 Line Chart
- 🔵 Scatter Plot

These visualizations helped identify trends, compare departmental performance, and understand the relationship between study hours, attendance, and marks.
